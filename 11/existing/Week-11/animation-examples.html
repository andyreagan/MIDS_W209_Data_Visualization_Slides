Example Animated Visualizations

gnuTellaVision: Real Time Visualization of a Peer to Peer Network
Rachna Dhamija, Danyel Fisher, Ka-Ping Yee (UC Berkeley)
Network analysis visualization, interaction assisted by animation

GapMinder World (Trendalyzer)
Hans Rosling (Gapminder Foundation)
Influential example of animation for time series. Be sure to hit the Play button. You may also wish to view the TED talk.

The Secret Lives of Numbers
Golan Levin, et. al. (for New Radio and Performing Arts, Inc.)
This one takes some effort, but it's a nice example of animation working with interaction. Note: this is a java applet. You may need to change your java security settings if viewing it on a Mac. (Go to System Preferences/Java/Security, enable java content in the browser, set security level to Medium).

Every Icon
John F. Simon Jr.
This one is more art than information, but we'll see it again in the aesthetics module.

Birth and Death
Samuel Huron, Romain Vuillemot, and Jean-Daniel Fekete (INRIA)
An example of the Visual Sedimentation library, which enables a particular animation effect that can be applied (with caution) to many graph types.

Org Org Chart
Justin Matejka (Autodesk)
Here an animation reveals complex organizational changes over time

Meso-to Planetary Scale Processes in a Global Ultra-High-Resolution Climate Model
Justin Small, Julio Bacmeister, David Bailey, et al. (NCAR)
This is a scientific visualization, so not technically within the scope of this class, but it's interesting to see how animation gets used in this community. The annual Supercomputing (SC) conference has a visualization showcase, and almost every entry is animated. This one, from SC13, is mesmerizing.
